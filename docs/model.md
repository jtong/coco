## 基于引用的复杂度

——《软件之道：软件开发争议问题剖析》 

| 指标             | 理想值   | FreeBSD   | Linux     | Solaris   | WRK       |
|-----------------|---------|-----------|-----------|-----------|-----------|
| 每个目录的文件数   |         | 6.8       | 20.4      | 8.9       | 15.9      |
| C 源文件的头文件数 |         | 1.05      | 1.96      | 1.09      | 1.92      |
|文件的平均结构复杂度 |         | 2.2x10^14 | 1.3x10^13 | 5.4x19^12 | 2.6x10^13 |

文件的平均结构复杂度 = （被引用数 x 引用数） ^ 2
